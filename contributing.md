# contributing.md

### Prerequisites

- [Python 3.8+](https://python.org)
- [Pipenv](https://github.com/pypa/pipenv)
- [Aero Task Runner](https://github.com/X-Store-App/aero)

### Running and building

#### In development
```
aero
```

#### When building
```
aero build
```
